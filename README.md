# Doctor Patient Portal

Doctor Patient Portal is a healthcare management web application developed to simplify interactions between doctors, patients, and administrators. The system provides an online platform for appointment scheduling, patient management, doctor scheduling, and healthcare administration.

## Project Overview
The application helps reduce manual hospital processes by digitizing appointment booking and patient record management. Patients can register, search doctors, and book appointments, while doctors can manage schedules and patient appointments. Administrators monitor the overall system.

## Features
- Patient Registration and Login
- Doctor Registration and Dashboard
- Appointment Booking System
- Doctor Schedule Management
- Patient Record Management
- Admin Panel for System Control
- Secure Authentication System
- User-Friendly Interface

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- JavaScript

## Modules
### Patient Module
- Create account
- Book appointments
- View doctors
- Manage profile

### Doctor Module
- View appointments
- Manage schedules
- Access patient information

### Admin Module
- Manage doctors
- Manage patients
- Monitor appointments
- System administration

## Purpose
The objective of this project is to provide an efficient healthcare management solution that improves communication between doctors and patients while reducing paperwork and manual scheduling processes.
